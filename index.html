Description

BedFix is a Minecraft Forge mod that removes the "You can only sleep at night" message and allows you to sleep in beds whenever you want.

You will no longer have to right-click your bed hundreds of times, waiting for the sun to be at just the right position where the game would finally allow you to sleep. Just right-click the bed once, at any time, and you will be sent to sleep immediately.

It works both in singleplayer and multiplayer, so long as the server has the mod installed as well.

BedFix is currently available for Minecraft 1.10.2, 1.11.2, 1.12, 1.12.1 and 1.12.2.

How To Install

BedFix can be installed just like any other Forge mod:

  1. Download and install Minecraft Forge.

  2. Place BedFix in the "mods" folder.
